##  Overview
**Animal Classification Game** is a fun and interactive web-based application built using **Flask**, where users test their knowledge by identifying animals from images.  
The app combines **AI-based classification** with a **gamified experience**, encouraging users to learn while they play.

---

## Features
-  Upload animal images to classify them using AI.  
-  Play the interactive guessing game and earn points.  
-  User badges system to reward top performers.  
-  SQLite database integration to store game progress and user data.  
-  Secure login, registration, and logout functionality.  
-  Responsive Flask web interface.

| Component | Technology |

| **AI Model** | Fastai / PyTorch |
